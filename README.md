    Advanced Archive Password Recovery (ARCHPR) 是一个
灵活的，适用于 ZIP 和 RAR 档案的高度优化的口令恢复工具。
它可以恢复保护口令或将用所有流行的档案版本创建的加密
ZIP 和 RAR 档案解除锁定。

    支持所有版本的 ZIP/PKZip/WinZip、RAR/WinRAR 以及
ARJ/WinARJ 和 ACE/WinACE (1.x)
